HTML (від англійської HyperText Markup Language) - це мова гіпертекстової розмітки тексту. Він потрібний, щоб розміщувати на веб-сторінки елементів: текст, зображення, таблиці та відео. Коли ви заходьте на сайт, браузер підвантажує HTML-файл з інформацією про структурі та контенті веб-сторінки.
CSS (англ. Cascading Style Sheets "каскадні таблиці стилів") - формальна мова декодування та опису зовнішнього вигляду документа (Веб-сторінки), написаного з використанням мови розмітки (частіше всього HTML чи XHTML). Також може застосовуватися до будь-яких XML-документи, наприклад, до SVG або XUL.
React. js - це бібліотека для мови програмування JavaScript з відкритим вихідним кодом для розробки інтерфейсів користувача. Вона допомагає швидко і легко реалізувати реактивність. коли у відповідь на зміну одного елемента змінюється решта.
Angular – це потужний фреймворк від Google, який використовується для створення мобільних програм. Однією з ключових особливостей Angular є те, що він використовує як мову програмування TypeScript, що також дозволяє підвищити читання коду і робити менше помилок під час
Vue. js - це прогресивний фреймворк для розробки інтерфейсів і односторінкових веб-додатків на мовою JavaScript. Він вирішує завдання рівня подання (view) та полегшує роботу з бібліотеками.
